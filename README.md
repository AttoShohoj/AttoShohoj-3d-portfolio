# AttoShohoj 3D Portfolio

An interactive and modern 3D portfolio web application designed to showcase `.glb` 3D models directly in the browser using WebGL-based rendering.

This project provides a lightweight and responsive solution for viewing 3D models with smooth interaction such as rotation, zoom, and real-time exploration.

---

## ✨ Overview

AttoShohoj 3D Portfolio is built to simplify 3D model visualization on the web. It allows users to present and interact with 3D assets without requiring any external software installation.

---

## 🎯 Key Features

- Real-time interactive 3D model viewer  
- Smooth rotation and zoom controls  
- Responsive design for all devices  
- Lightweight and fast loading performance  
- Browser-based rendering (no installation required)  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- WebGL  
- model-viewer (3D rendering engine)

---

## 📦 Supported Format

- `.glb` (Recommended)

---

## 🚀 Purpose

This project is created for learning and showcasing 3D web development concepts and building a personal 3D portfolio experience.
